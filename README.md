# Explainable AI Dashboard

An enterprise-grade, institutional decision-support platform for stock analysis and prediction, featuring:
- **LSTM & Transformer Models**: Advanced AI for price prediction.
- **Explainability**: Insights into model confidence and decision factors.
- **Risk Heatmap**: Visualization of volatility vs. confidence.
- **Institutional Design**: Sleek, high-performance UI tailored for professional traders.

## Tech Stack
- **Frontend**: React, Vite, TailwindCSS (Radix UI)
- **Backend**: Node.js, Express, MongoDB
- **Data**: Python (for real-data fetching and processing)

## Getting Started

### Prerequisites
- Node.js
- MongoDB
- Python 3.x

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   cd server && npm install
   ```
3. Set up your `.env` variables in the `server` folder.
4. Run the development server:
   ```bash
   npm run dev
   ```
